# 脑洞组装工坊

一个基于 Vue 3 的互动式小说创作工具。

## 核心流程

1. 输入故事起点，只提供开场设定，不直接提供完整脑洞。
2. 结合风向标和导语数据库，由 AI 生成可延展的故事脑洞。
3. 生成导语，并让导语与第一个剧情点自然衔接。
4. 按章节推进剧情。
5. 每章包含 4 个剧情点，每个剧情点提供 4 个分支选项供用户选择。
6. 每章结束后生成 4 个章节钩子，用户选择一个决定下一章方向。
7. 第四章结束后额外生成 4 个大钩子，作为关键转折点。
8. 完整剧情组装完成后，再输入指定文风，由 AI 输出最终成文。

## 设计原则

- 不预设固定付费点或强行卡点。
- 依靠用户选择驱动剧情发散。
- 让剧情组装和最终写作分成两个阶段。

## 开发

```bash
npm install
npm run dev
```

构建生产版本：

```bash
npm run build
```
